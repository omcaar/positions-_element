Das gute alte CSS Position Element

Was kann es?

  • Elmente aus dem restlichen Elementfluss entfernen
  • an jeder beliebigen Stelle positioniert werden

Verschiedenen Möglichkeiten

  • absolute
    • positioniereung an einer Stelle die durch left right top und bottem festgelegt wird

      • Beispiel absolut

  • fixed
    • ist gut für Fußzeilen

      • Beispiel fixed

  • relative
    • Element richtet sich an den Elementen "neben" sich aus
    • unterbricht aber den textfluss

      • Beispiel relative

  • static
    • das Element bleibt da wo es eh schon ist
    • bleibt also im Elementfluss

      • Beispiel static

  • sticky
    • Element behält seine Position im Elementfluss
    • bis ans obere oder untere Seitenende dort „kleben“ bleiben.
